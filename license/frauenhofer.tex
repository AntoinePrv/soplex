\begin{center}

\begin{center}
Fraunhofer-Institute for Algorithms

Schloss Birlinghoven, 53754 Sankt Augustin
\end{center}
in folgenden Lizenznehmer genannt, 
über das Softwarepacket SoPlex im folgenden Software gennant:
}

\begin{enumerate}

\item Das ZIB erteilt dem Lizenznehmer eine zeitlich unbefristete, nicht
   ausschließliche Lizenz zur Benutzung und Verwertung des
   Softwarepacketes SoPlex. 

\item Das Eigentumsrecht an der Software verbleibt beim ZIB.

\item Hiermit bestätigt der Lizenznehmer, das er die Software
   vereinbarungsgemäß vollständig im Quellcode erhalten hat.

\item Diese Lizenz gilt auch für alle nachfolgenden Versionen der
   Software die auf der Website des ZIB unter der ZIB Academic License
   bereit gestellt werden.

\item Das ZIB überläßt dem Lizenznehmer die Software \glqq so, wie sie
   ist\grqq. 

   Gewährleistungsansprüche an das ZIB sind ausgeschlossen.

   Es besteht keinerlei Verpflichtung von seiten des ZIB mögliche
   Fehler die beim Gebrauch der Software auftreten zu korrigeren.
   Es wird keinerlei Garantie für Funktionsfähigkeit und Eignung für
   einen bestimmten Zweck gegeben. 

\item Das Urheberrecht für die Software verbleibt beim ZIB. 

\item Der Lizenznehmer hat das Recht, für den internen Gebrauch eine
   beliebige Anzahl von Kopien zu erstellen und die Software als
   sogennante Bibliothek in binärer Form an  Softwareprodukte zu
   binden und einzusetzen. 

\item Das ZIB übernimmt keine Haftung für Schäden jeglicher Art, die
   sich aus der Verwendung der Software ergeben. 

\item Lizenznehmer zahlt für die Lizenz nach Rechnungstellung durch
   das ZIB einen einmaligen Kaufpreis in Höhe von Euro 18.000
   zuzüglich der gesetzlichen Mehrwertsteuer.
 
\item Als Gerichtsstand ist Berlin vereinbart.

\end{enumerate}
