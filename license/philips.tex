\begin{enumerate}

\item Das ZIB erteilt dem Lizenznehmer eine nicht-ausschließliche, zeitlich,
   räumlich und inhaltlich unbeschränkte, unwiderrufliche und nicht
   über\-trag\-bare 
   Lizenz zur Benutzung des Softwarepaketes SoPlex zu nachstehenden
   Bedingungen:

   Die Software darf ausschließlich zu internen Zwecken des
   Lizenznehmers beim Lizenznehmer eingesetzt werden.

   Die Lizenz erlaubt maximal die gleichzeitige Installation und Verwendung 
   von zehn (10) Kopien der Software beim Lizenznehmer.

   Eine Weitergabe der Software an Dritte, ist in jeglicher
   Form unzulässig. Die Weitergabe von Lizenzen oder die 
   Vergabe von Unterlizenzen ist ebenfalls unzulässig. 
   
\item Das ZIB gewährleistet, die Software selbst entwickelt zu haben und dass
     die nach diesem Vertrag eingeräumten Nutzungsrechte des Lizenznehmers
     keine Rechte Dritter verletzen.

     Alle hierüber hinausgehenden Gewährleistungsansprüche an das 
     ZIB sind ausgeschlossen.

\item Diese Lizenz gilt auch für alle nachfolgenden Versionen der
   Software, die vom ZIB frei gegeben werden.

\item Bei Vertragsabschluß gilt die Software als vollständig übergeben.

\item Der Lizenznehmer zahlt für die Lizenz nach Rechnungstellung durch
   das ZIB eine einmalige Lizenzgebühr in Höhe von Euro 1.000,00
   zu\-züg\-lich der gesetzlichen Mehrwertsteuer.

\item Das Eigentumsrecht an der Software verbleibt beim ZIB.

\item Das Urheberrecht für die Software verbleibt beim ZIB. 

\item Das ZIB überläßt dem Lizenznehmer die Software \glqq so, wie sie
   ist\grqq und steht nicht dafür ein, dass die Software für die
   Zwecke des Lizenznehmers brauchbar ist.

\item Das ZIB übernimmt keine Wartungsverpflichtungen.

\item Das ZIB übernimmt keine Haftung für Schäden jeglicher Art, die
   sich aus der Verwendung der Software ergeben. 
   Dies gilt nicht für Vorsatz oder
   grobe Fahrlässigkeit. Für diesen Fall wird die Höhe der Haftungssumme
   beschränkt auf den Betrag gemäß Ziffer 5.

\item Sollten einzelne Bestimmungen dieser Vereinbarung unwirksam sein
   oder werden, so wird dadurch die Gültigkeit der übrigen Bestimmungen
   nicht berührt. Die unwirksamen Bestimmungen sind in diesem Fall durch
   Regelungen zu ersetzen, die dem wirtschaftlichen und rechtlichen Zweck
   der unwirksamen Bestimmungen am nächsten kommen.

\item Als Gerichtsstand ist Berlin vereinbart.

\end{enumerate}
