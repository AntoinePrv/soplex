Notes for SoPlex Release 1.5

What's new
----------
 - major rework of preprocessing routines:
   + several bugfixes concerning basis statuses, dual values
     and reduced costs
   + improved performance in linear dependency check:
     exclude singleton rows and columns
 - improved handling of singular bases and cycling
 - several bugfixes and improvements in numerical stability

SoPlex interfaces
-----------------
 - The LP interface of SCIP 2.0 to SoPlex now uses
   equilibrium scaling when solving without advanced
   starting basis.
 - We have updated the SoPlex interface contained in the
   OSI project of the COIN-OR initiative.

For details see the CHANGELOG file.
